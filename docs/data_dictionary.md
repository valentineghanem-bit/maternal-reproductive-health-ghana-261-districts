# Data Dictionary — `master_maternal_ghana_261districts_v1.csv`

Rows: 261 (districts; Guan, Oti Region, included — see `project_ghana_districts_count.md`)

Version: v1.0 | Extraction year: 2022

`data_source_*` column (32 total) provides row-level provenance for its corresponding
substantive column and satisfies EX-003 (data source attribution).

## 6. Machine Learning Outputs (Phase 8)

|---|---|---|
| `xgb_predicted_risk` | string | Predicted risk class (HIGH / MODERATE / LOW) from the gradient-boosted-tree model. **Per ML-NAMING-001 (methodology.md §7.2): the model is `GradientBoostingClassifier` (scikit-learn), the documented sklearn fallback used because XGBoost was unavailable in the execution environment. **Not present in v1.0 (reserved for future phases):** `xgb_prob_high`, `xgb_prob_moderate`,

per-class probability outputs is a possible future-phase enhancement, not an error.

## 7. Data Source Attribution Columns (`data_source_*`, EX-003)

Each substantive column above (where applicable) has a corresponding `data_source_<column>`
string column recording its provenance, e.g. `"Ghana DHS 2022 (FR387), regional"`,
`"GSS Census 2021, district"`, or a `"Derived (...)"` description for computed columns
(`composite_maternal_index`, `gwr_local_r2`). `data_source_spatial_weights` documents the
spatial weights matrices used (KNN k=4 for Moran's I; Rook contiguity for LISA; see
`Ghana_New_260_District.geojson`). 32 `data_source_*` columns are present, satisfying the
`test_data_source_columns` canonical test (≥4 required).
